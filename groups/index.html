Note that the place for meetings changes for many of the user groups, so always check the user group page for the most up-to-date information. View F# User Groups in a larger map. If you are running an F# user group that is missing to the list, please edit this page!

Worldwide

Community for F#

The Community for F# is a virtual community for discussing and learning about F#. If you would like to stay up-to-date with news and announcements, please join our Live Group and follow us on Twitter. Also, please give us feedback!

For more information, see the group page on FPish.

North America

The San Francisco Bay Area F# User Group

SF# is a group of developers in the San Francisco Bay Area who have an interest in the F# programming language. Novices and experts are both welcome.

For more information, see the group page on meetup.

Detroit F# Meetup

A group of developers in Motown who are interested in learning to use F# to write better code. We meet once a month.

For more information, check the Detroit F# Meetup page.

Cleveland, Ohio F# SIG

The Cleveland F# SIG gives programmers at all skill levels a monthly opportunity to learn about the language and how others are using it, improve their skills and share their knowledge in a relaxed, cooperative environment.

For more information, visit the group web site.

Houston F# User Group

A group for developers in the Houston area with an interest in the F# programming language. Programmers of all skill levels and backgrounds are welcome.

For more information, see the group page on meetup.

New England F# Users Group

This group is for anyone interested in the F# programming language. We'll discuss a wide variety of topics related to programming with F#, and developers of all skill levels and backgrounds are welcome.

For more information, see the group page on meetup.

Chicago F# Users

The Chicago F# Users group is dedicated to the advancement of the F# language throughout the Chicago area. If you are curious (or skeptical) about the F# language, regardless of your level of expertise with it, you are welcome! We would like to meet regularly to present solutions to real world problems that can be addressed through the use of F#.

For more information, see the group page on meetup.

New York City F# User Group

A group for both learning functional programming and exploring the limits of the F# programming language. Programmers of all skill levels and backgrounds are welcome.

For more information, see the group page on meetup.

F# Seattle User Group

A group for learning and promoting F# programming language. Programmers of all skill levels and backgrounds are welcome.

For more information, see the group page on meetup.

Functional Programming Phoenix Group

Developer group for functional languages (F# and others) and functional programming paradigms

For more information, see the group page on meetup.

Washington D.C. F# Meetup

This is an F# meetup group that meets in and around the DC area. The goal is to bring likeminded local F# enthusiasts together to learn, play, and explore the F# language and ecosystem. Programmers of all skill levels and backgrounds are welcome.

For more information, see the group page on meetup.

Europe

Zurich F# Users

Our goal is to promote the usage of the multi-paradigm programming language F# in Switzerland. IT experts who have a professional or personal interest are cordially invited to join the group. We speak German and English.

For more information, see the group page on meetup.

Functional Programming in F#

Learn how to take advantage of F# and .NET platform to develop a real world functional programs If you are beginner or an expert in FP it doesn't matter. You are welcome!

For more information, see the group page on meetup.

Minsk F# User Group

This group is for everyone interested in functional programming and F# programming language.

For more information, see the group page on meetup.

Athens F# User Group

Our goal is to promote the usage of functional programming and especially F# (FSharp) in Greece.

For more information, see the group page on meetup.

The Greater Helsinki Area F# User Group

This is a group of developers in the Greater Helsinki Area (Helsinki, Espoo, Vantaa, etc.), Finland, who have an interest in the F# programming language. Novices and experts are both welcome.

For more information, see the group page on meetup.

Functional Programming Stockholm

A user group focusing on functional programming in all of its forms. We meet at different locations in the beautiful city of Stockholm to discuss what we enjoy - functional programming.

For more information, see the group mailing list.

F#unctional Londoners Meetup Group

F#unctional Londoners is a meetup group that aims to bring together Londoners with an interest in functional programming with F#.

For more information, see the group page on meetup.

Asia

F# User Group Japan

A user group for the F# programming language - a functional-first multi-paradigm programming language.

This community has been mainly a virtual Google group. Please register to get all important information and share your updates - be it up-to-date information, questions or other topics related to F#.

F# User Group China

A virtual user group for Chinese F# programmer. The community uses QQ to exchange F# programming ideas in real time. The QQ group number is 61436709, which can be found by following the instructions. Please join and share up-to-date information, questions, and tips related to F#.

Australia

F# User Group Sydney

A monthly user group for F# people living in Sydney. Please join our Meetup group to be notified of our next meeting and to get involved.
